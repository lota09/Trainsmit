# 스마트 대중교통 도착정보 디스플레이 프로젝트

## 프로젝트 개요

본 프로젝트는 아두이노 또는 ESP32 등의 임베디드 보드를 활용하여 사람의 움직임을 감지하고, 실시간으로 지하철 도착정보를 표시하는 스마트 디스플레이 장치를 개발하는 것을 목표로 합니다.

## 주요 기능

### 1. 사람 감지 기능
- 적외선 센서(PIR) 또는 움직임 감지 센서를 통한 사람 인식
- 센서 감지 시 자동으로 교통정보 모드로 전환

### 2. 실시간 교통정보 제공
- WiFi 연결을 통한 실시간 지하철 도착정보 API 호출
- 사용자가 미리 등록한 지하철역의 도착정보 표시
- 노선별, 방향별 도착 예정시간 및 열차 정보 제공

### 3. 디스플레이 기능
- 8인치 이상의 컬러 디스플레이를 통한 GUI 렌더링
- 직관적이고 가독성 높은 교통정보 화면 구성
- 대기 모드 시 시계 화면 표시

### 4. 운행시간 관리
- 지하철 운행 종료 시간 확인 기능
- 운행 종료 시간 이후 적절한 대체 화면 제공

## 시스템 동작 흐름

```
시작 → 센서 대기 → 사람 감지? 
                    ↓ (예)
                운행시간 확인 → 운행중?
                               ↓ (예)
                         WiFi 연결 → API 호출 → 정보 표시
                               ↓ (아니오)
                         운행종료 화면 표시
                    ↓ (아니오)
                시계 화면 표시
```

## 예상 하드웨어 구성

### 필수 부품
- **메인 보드**: ESP32 (WiFi 내장) 또는 Arduino + WiFi 모듈
- **센서**: PIR 모션 센서 또는 적외선 거리 센서
- **디스플레이**: 8인치 이상 TFT LCD 또는 OLED (SPI/I2C 통신)
- **전원부**: 5V/12V 어댑터 및 전압 레귤레이터
- **케이스**: 3D 프린팅 전용 설계 케이스

### 선택 부품
- 터치 스크린 (사용자 설정 인터페이스용)
- 스피커 (알림음 또는 음성 안내용)
- LED 표시등 (상태 표시용)

## 기술 스택

### 임베디드 개발
- **개발 환경**: Arduino IDE 또는 PlatformIO
- **프로그래밍 언어**: C/C++
- **통신 프로토콜**: WiFi, HTTP/HTTPS, JSON 파싱
- **디스플레이 라이브러리**: TFT_eSPI, Adafruit GFX, LVGL

### API 연동
- **교통정보 API**: 
  - 서울교통공사 지하철 실시간 도착정보 API
  - 공공데이터포털 대중교통 API
- **시간 동기화**: NTP 서버 연동

### 디자인 및 제작
- **3D 모델링**: Fusion 360, Tinkercad
- **3D 프린팅**: PLA/PETG 소재
- **GUI 디자인**: 픽셀 아트 또는 벡터 그래픽
